離線票券管家 (V3.0.1)

這是一個基於瀏覽器本地儲存的輕量級票券管理應用程式。它旨在提供一個無需網路連線、注重隱私且功能強大的工具，用於整理、追蹤和核銷各種電子票券、序號或實體卡券。

核心特色

完全離線：所有資料皆儲存在瀏覽器本地（LocalStorage），無需伺服器，確保數據隱私和存取速度。

智慧辨識與批量新增：支援從簡訊或筆記中智慧解析多行文字，自動提取「品名」、「序號」和「有效期限」，並可批量匯入。

多圖管理與圖庫：單張票券可附加多張圖片（如截圖），並可將常用圖片加入個人圖庫，支援剪貼簿貼上及網址導入。

序號視覺化：可將票券序號即時生成 QR Code 和 Barcode 128 條碼，方便掃描核銷。

多維度整理：

標籤 (Tags)：支援多重標籤分類，方便篩選和整理。

期限追蹤：自動根據效期排序，並有視覺化儀表板與紅字提醒。

資料安全與備份：提供 JSON 格式的資料匯出與匯入功能，方便使用者在不同瀏覽器或設備間備份和遷移資料。

Telegram 通知 (選配)：可選配設定 Telegram Bot Token 和 Chat ID，在票券即將到期時發送每日提醒訊息。

技術棧

前端框架：React 18 (使用 Babel 在瀏覽器內即時編譯)

樣式與 UI：Tailwind CSS (透過 CDN 引入)

條碼/二維碼：

Barcode：bwip-js

QR Code：qrious

狀態與儲存：瀏覽器 localStorage（用於永久儲存）

環境建置與使用

由於這是一個單一 HTML 檔案應用程式，您不需要任何複雜的建置環境或伺服器。

1. 運行應用程式

只需將 新文字文件.HTML 檔案下載到您的電腦或手機。

使用任何現代瀏覽器（Chrome, Firefox, Safari 等）直接雙擊開啟此檔案即可運行。

2. 資料存取

首次開啟時，您需要設定一個清單名稱 (Workspace ID)，例如 MyTickets。所有資料將綁定到此 ID，儲存在您瀏覽器的本地儲存空間中。

更換瀏覽器或清除瀏覽器快取時，請務必先透過「備份還原」功能匯出資料。

3. 如何新增票券

點擊右下角的 + 按鈕開啟新增彈窗。

選擇 「智慧貼上 (批量)」：將您從簡訊、Email 或聊天記錄中複製的多行文字貼上，系統將自動解析序號、品名和期限。您可以在下方設定統一的「品名」或「標籤」來覆蓋解析結果。

選擇 「手動輸入」：適用於單張、格式不規則的票券。

4. 圖片管理

在新增/編輯票券時，您可以點擊「上傳」、「貼上」或「圖庫」來添加圖片。

貼上圖片功能支援直接從剪貼簿讀取圖片檔案。

圖片壓縮：所有匯入的圖片都會被壓縮到最大 800x800 像素並轉換為 JPEG 格式（品質 60%），以節省本地儲存空間。

5. 資料備份與還原

點擊頂部菜單的「備份還原」圖示。

匯出：建議選擇「全匯出 (JSON 檔案)」，下載包含所有票券數據（含圖片 Base64 編碼）的 JSON 檔案。

匯入：可選擇「上傳 JSON 檔案」或「貼上文字」，並選擇 附加（保留現有數據並加入新數據）或 覆蓋（刪除現有數據）。
