🎫 離線票券管家 V2 (Offline Ticket Manager)

簡單、隱私、無伺服器的個人電子票券管理工具。

離線票券管家 是一個單一 HTML 檔案的 Web 應用程式，旨在解決電子票券（優惠券、兌換券、訂房憑證）散落在各個 Email、簡訊或 App 的問題。V2 版本帶來了更強大的圖片管理與智慧輸入功能。

✨ 特色功能

🔐 100% 隱私與離線儲存

無後端伺服器：所有資料（包含圖片）皆儲存在您瀏覽器的 localStorage 中。

資料不外流：您的票券資訊永遠不會上傳到雲端，完全掌握在自己手中。

🤖 智慧輸入助手 (V2 新增)

AI 規則解析：直接貼上簡訊或 Email 內容，系統自動嘗試抓取「標題」、「取得日期」與「到期日期」。

寬版介面：優化的輸入視窗，方便編輯大量文字。

📸 進階圖庫管理 (V2 新增)

多圖支援：單一票券可上傳多張截圖或照片。

自動壓縮：內建圖片壓縮演算法，避免瀏覽器儲存空間過快耗盡。

封面設定：可自由選擇其中一張圖片作為列表卡片的封面主圖。

📂 完整的管理功能

分類系統：食、衣、住、行、育、樂、其他。

狀態追蹤：自動標示「快過期」、「已過期」或「已使用」。

搜尋過濾：即時搜尋標題與備註內容。

下載本專案中的 OfflineTicketManagerV2.html 檔案。

直接使用現代瀏覽器（Chrome, Edge, Safari, Firefox）開啟該檔案。

開始使用！

使用教學

新增票券：點擊右下角的 + 按鈕。

匯入內容：

手動輸入：填寫名稱、日期與分類。

智慧輸入：點擊「智慧輸入」，貼上收到的票券簡訊，讓系統自動填寫。

圖片管理：點擊上傳區塊選取圖片（支援多選），上傳後點擊星星圖示 ⭐ 設定為主圖。

搜尋：在上方搜尋列輸入關鍵字即可快速找到票券。

🛠️ 技術棧

核心：原生 HTML5, JavaScript (Vanilla ES6+)

樣式：Tailwind CSS (透過 CDN)

圖標：Font Awesome (透過 CDN)

儲存：Browser LocalStorage API

⚠️ 注意事項

資料安全：由於資料儲存在瀏覽器的 LocalStorage 中，清除瀏覽器快取/資料將會導致票券資料遺失。建議定期備份重要的票券截圖。

儲存空間限制：瀏覽器通常對 LocalStorage 有 5MB~10MB 的容量限制。本工具已內建圖片壓縮，但若存入過多高畫質圖片仍可能達到上限。

網路需求：雖然資料是離線儲存，但開啟時需要網路連接以載入 Tailwind CSS 與 Font Awesome 的樣式資源（CDN）。
